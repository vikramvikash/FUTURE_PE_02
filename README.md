## Overview

This repository contains a complete AI-generated **UGC (User Generated Content) Ad Content Pack** for **Notion** — the all-in-one productivity SaaS tool.

All content was created using structured prompt engineering on Claude (claude.ai) and ChatGPT, following real-world UGC ad frameworks used by content marketing agencies.

---

## Product Chosen

**Notion** — All-in-one workspace for notes, tasks, wikis, and databases.  
🔗 https://www.notion.so

**Why Notion?**  
Notion has a massive, relatable target audience (students, freelancers, remote teams) with clear pain points — making it ideal for authentic UGC-style ad content.

**Target Audiences:**
- Students managing assignments and projects
- Freelancers juggling multiple clients
- Remote teams struggling with scattered tools
