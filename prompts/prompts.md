# Structured Prompts — Notion UGC Ad Content Pack

> These prompts use `[bracketed variables]` and can be reused for any product or platform.

---

## Prompt 01 — Hook Generator

**Purpose:** Generate multiple scroll-stopping opening lines for a UGC ad.

```
You are a UGC ad copywriter for [Product: Notion].

Write 3 scroll-stopping hooks for [Platform: Instagram Reel].

Target audience: [freelancers / students / remote teams].

Rules:
- Sound like a real person talking, not a brand ad
- No emojis
- Keep each hook under 15 words
- Focus on the pain point, not the product features
- Each hook must feel different (POV, question, confession, stat)

Output format: numbered list, one hook per line.
```

**Variables to swap:**
- `[Product]` → any SaaS tool, D2C brand, or local business
- `[Platform]` → Instagram Reel / YouTube Shorts / Facebook Ad
- `[Target audience]` → students / freelancers / remote teams / small business owners

## Prompt 02 — Full Ad Script Generator

**Purpose:** Generate a complete short-form UGC ad script using the problem → solution → CTA framework.

```
You are a UGC content creator writing a short-form ad script for [Product: Notion].

Use this exact structure:
1. Hook (1 sentence — relatable pain point or POV moment)
2. Problem (2–3 sentences — expand the pain, make the viewer feel seen)
3. Solution (2–3 sentences — how Notion solves it, no jargon, no corporate language)
4. CTA (1 sentence — soft, conversational, not salesy)

Tone: casual, honest, first-person narrative
Style: UGC — sounds like a real user, not a spokesperson
No marketing buzzwords. No exclamation marks. No emojis.

Target audience: [freelancers managing multiple clients]
Platform: [Instagram Reel]
Video length target: [30–45 seconds]
```

**Variables to swap:**
- `[Product]` → any product or service
- `[Target audience]` → define who the viewer is
- `[Platform]` → changes tone and length
- `[Video length target]` → 15 sec / 30–45 sec / 60 sec

## Prompt 03 — Platform Adapter

**Purpose:** Take an existing script and adapt it for a different platform and audience.

```
Here is a UGC ad script:
[Paste your script here]

Adapt this script for [Target Platform: Facebook Ad].

Make these specific adjustments:
- Facebook audience skews older (25–45), more text-tolerant
- Strengthen the benefit statement within the first 3 seconds
- Keep total spoken length under 60 seconds
- CTA must mention the free plan explicitly
- Maintain the authentic UGC tone — do not make it sound like a traditional ad

Output the adapted script only. Do not explain changes.
```

**Variables to swap:**
- `[Target Platform]` → Facebook Ad / YouTube Shorts / LinkedIn / Twitter/X
- Adjust the audience notes and CTA requirements per platform
